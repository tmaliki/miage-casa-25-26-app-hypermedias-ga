Mémento CSS

Ce tableau récapitule les principales propriétés CSS utilisées dans le TP CSS3.

Propriété Description Exemple d'utilisation
color Couleur du texte color: red;
background-color Couleur de fond background-color: #eee;
margin Marge extérieure margin: 20px;
padding Espacement intérieur padding: 15px;
border Bordure du bloc border: 2px solid black;
display Type d'affiche (block, inline, flex, ...) display: flex;
font-size Taille du texte font-size: 18px;
font-family Police du texte font-family: Arial, sans-serif;
border-radius Arrondi les coins d'un bloc border-radius: 10px;
box-shadow Ombre portée d'un bloc box-shadow: 4px 4px 10px rgba(0, 0, 0, 0.5);
text-shadow Ombre du texte text-shadow: 2px 2px 4px gray;
justify-content Alignement horizontal des éléments Flexbox justify-content: space-between;
align-items Alignement vertical des éléments Flexbox align-items: center;
flex-direction Direction des éléments Flexbox flex-direction: column;
@media Règle pour le responsive design (Media Queries) @media screen and (max-width: 480px) { ... }
